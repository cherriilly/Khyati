TinDog

Tindog is a fun and engaging web development project designed as a Tinder-inspired Website design for dogs. Built using HTML, CSS, and Bootstrap, this responsive webpage adapts seamlessly across devices. It showcases clean design and modern web technologies, creating an inviting experience for dog lovers looking to connect their furry friends.

Move It

Move It is a vibrant web development project designed as a homepage for a moving company. Created with HTML, CSS, and Bootstrap, this responsive webpage ensures smooth functionality across devices. Its modern design and user-friendly interface embody the company’s goal of making moving effortless and enjoyable.

Art Meets Code: Piet Mondrian-Inspired Creations

Piet Mondrian-inspired project replicating the iconic artworks 'Composition No. III, with Red, Blue, Yellow, and Black' and 'Composition with Red, Blue, and Yellow.' Built using HTML and CSS, the project leverages the Grids concept to recreate Mondrian’s geometric precision and bold use of primary colors in a web format.
